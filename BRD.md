> 작성일: 2026-05-19

## 1. 프로젝트 개요

### 1.1 배경
씨젠 의료사업부 이무팀(5명: 김성일 팀장 / 이한표 차장 / 강경모 과장 / 노용철 대리 / 유다빈 대리)은 매주 **주간업무 보고서(Word 문서)** 를 작성해야 한다.
- 기존에는 팀원 각자 Word에 직접 표를 그려 작성 → 양식이 매번 흐트러지고, 차주 계획을 다음 주차 금주 업무로 옮겨 적는 작업이 반복됨.
- 업무 진척 상황(긴급/예정/진행중/완료)을 한 차원에서 확인하는 도구가 부재.
- 외부 SaaS(Notion, Jira 등) 대신 보안·비용의 어려움 → **사내 PC 브라우저에서 단일 HTML 파일로 동작하는 가벼운 도구**가 필요함.

### 1.2 목적 (Business Goals)
1. **주간업무 보고서 작성 시간 단축** — 팀원 1인당 주비 1시간 이상 소요되는 작성·정리 시간을 절반 이하로.
2. **양식 통일** — 팀 전체가 동일한 Word 형식("구분 / 금주 업무 / 일정 / 담당자 / 차주 업무 / 일정 / 담당자 / 비고")으로 자동 추출.
3. **업무 가시성 확보** — 칸반(긴급/예정/진행중/완료) 형태로 팀 전체 업무 현황을 대시보드에서 확인.
4. **차주 계획 → 금주 업무 자동 인계** — 지난주 작성한 "차주 업무"를 이번 주차 진입 시 자동 안내.

### 1.3 비-목표 (Out of Scope)
- 다중 사용자 실시간 동시 편집 (서버 없음, localStorage 기반).
- 로그인·권한관리·감사로그.
- 모바일 전용 UI (PC 크롬 1280px 이상 기준).
- 그룹웨어/메일/캘린더 연동.

## 3. 비즈니스 요구사항 (High-level)

### BR-1. 업무 등록·관리
- 팀원이 본인/팀원의 업무를 자유롭게 등록·수정·삭제할 수 있어야 한다.
- 한 업무에 **복수 담당자** 지정 가능.
- 상태: 긴급 / 예정 / 진행중 / 완료 (+ 내부기 지연/보류).
- 시작일·마감일은 "미정" 처리 가능.

### BR-2. 주간 단위 이력
- 모든 업무·보고서 데이터는 **ISO 주차(예: 2026-W20)** 단위로 분리 관리.
- 주차 이동 시 해당 주차 데이터만 표시.
- 달력 표시를 통해 임의 주차로 선택 가능.

### BR-3. 주간보고서 작성
- 팀원별로 "구분 / 금주 업무내용 / 일정 / 담당자 / 차주 업무내용 / 일정 / 담당자 / 비고" 8열 테이블을 직접 편집.
- 금주/차주 업무내용 셀에서 **리치텍스트** (볼드/밑줄/색상/표 삽입/이미지 삽입) 지원.
- 개인 보기 / 전체 보기 선택.
- **Word 추출** 버튼으로 사내 양식에 맞는 .doc 파일 다운로드 (씨젠  브랜드 컬러 #C4181F 적용).

### BR-4. 차주 → 금주 자동 인계
- 새 주차 진입 시, 이전 주차의 "차주 업무내용"을 모달로 안내 → 체크박스 선택 후 일괄 import.
- 별도로 "지난주 업무 불러오기" 버튼 제공 (기존 데이터는 보존, 추가만 됨).

### BR-5. 기간 조회·추출
- 한 팀원의 최근 2/3/4주차 보고서를 한 번에 조회 가능.
- 해당 기간을 한 Word 파일로 추출 (시간 보고용).

### BR-6. 데이터 보존·이동
- 모든 데이터는 브라우저 **localStorage** (`ga_erp_v1` 키)에 저장.
- JSON 내보내기/불러오기로 팀원 간 동기화·백업 (예: `씨젠 이무팀_2026-05-12.json`).
- 팀원·업무 분류는 설정 탭에서 직접 추가/삭제/직급 부여.

### BR-7. 디자인·브랜드
- 씨젠 의료사업부 CI 적색(#C4181F) 기반 컬러 시스템.
- 좌측 상단 씨젠  로고(SVG inline) 노출.
- 한글 가독성을 위해 Noto Sans KR + 맑은 고딕(Word) 사용.

## 5. 제약 조건

- **단일 HTML 파일** (`index.html`) — 외부 빌드 도구·서버·CDN 의존성 최소화. 폰트(Google Fonts)만 외부.
- **서버 없음** — 모든 상태는 클라이언트 localStorage. 5MB 한도 인지(이미지 삽입 시 경고).
- **사내 보안망 환경** 가정 — 외부 API 호출 금지.
- **MS Word .doc 호환** — `application/msword` MIME, MSO 네임스페이스 포함 HTML 방식으로 생성.
